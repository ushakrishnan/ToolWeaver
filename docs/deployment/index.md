# Deployment Guides

Use these runbooks to deploy ToolWeaver in different environments. Pick the guide that matches your stack and maturity level.

- [Production Deployment](production-deployment.md) — End-to-end hardening, performance, monitoring, and scaling.
- [Azure Setup](azure-setup.md) — Configure Azure Computer Vision OCR for receipt flows.
- [Redis Setup](redis-setup.md) — Caching options: local, WSL, Azure Cache, Redis Cloud.
- [Qdrant Setup](qdrant-setup.md) — Vector DB options: local, WSL, Qdrant Cloud, Azure ACI.
- [SQLite + Grafana / OTLP](sqlite-grafana-setup.md) — Analytics backends (SQLite, Prometheus, Grafana Cloud OTLP).
